# Products App

## Requirements
- `nodejs`

## Installation
1. Add the ff. to your `hosts` file:
   ```
   127.0.0.1 products-api.test
   ```
1. Clone this repository
   ```bash
   git clone https://github.com/juanitotrrj/yp-products-client.git && cd yp-products-client
   ```
1. Run the app
   ```bash
   npm start
   ```
1. View the app @ http://localhost:3000/
